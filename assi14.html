Task 2: Simple XML Document

Create a small XML structure to store data (e.g., books):

    

  
    Book One
    Author A
    2023
  

<?xml version="1.0" encoding="UTF-8"?>
<books>
    <book>
        <title>Your Book Title</title>
        <author>Author Name</author>
        <year>2025</year>
    </book>
</books>

Task 3: Compare Tag Rules

Update your HTML and XML examples to see these differences:

  • HTML tags are predefined; XML tags are custom.
  • HTML is not case-sensitive; XML is case-sensitive.
  • HTML allows optional closing tags; XML requires all tags closed.

Task 4: Add Attributes

Add attributes to your HTML and XML elements:

<img src="image.jpg" alt="Example Image"> <!-- HTML -->
<book id="b1">...</book> <!-- XML -->

Task 5: Validate Your XML

Check that your XML is well-formed:

  • All tags are properly closed.
  • Tag names match exactly (case-sensitive).
  • No overlapping tags.
